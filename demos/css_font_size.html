纯css实现字体的大中小

当我还只有六岁的时候，在一本描写原始森林的名叫《真实的故事》的书中， 看到了一副精彩的插画，画的是一条蟒蛇正在吞食一只大野兽。页头上就是那副画的摹本。

这本书中写道：“这些蟒蛇把它们的猎获物不加咀嚼地囫囵吞下，尔后就不 能再动弹了；它们就在长长的六个月的睡眠中消化这些食物。” 当时，我对丛林中的奇遇想得很多，于是，我也用彩色铅笔画出了我的第一副图画。

我把我的这副杰作拿给大人看，我问他们我的画是不是叫他们害怕。 他们回答我说：“一顶帽子有什么可怕的？”

我画的不是帽子，是一条巨蟒在消化着一头大象。于是我又把巨蟒肚子里的情况画了出来，以便让大人们能够看懂。这些大人总是需要解释。
